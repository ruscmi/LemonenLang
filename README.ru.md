# LemonLang
Приветствую. LemonLang - мой интерпретатор на С++ с простейшей логикой парсинга строк. В данный момент он достаточно простой в логике и работает достаточно быстро. Если вам понравился мой проект если хотите поставьте мне звезду,я буду очень рад этому.

# Установка
Сначала скопируйте клон `git clone https://github.com/ruscmi/LemonenLang.git` Перейдём в папку исходного кода на линукс дистрибутивах `~/src` или просто измените место, где находятся исходники, затем скомпилируйте все файлы и создайте lmnlang `g++ main.cpp logic.h lmnlang.cpp -o lmnlang` . Далее запускаем `./lmnlang` . Спасибо за использование моих исходников,вы можете их самостоятельно изменять и вносить что либо свое так как они полностью открыты.

Под Windows есть отдельный скомпилированный файл [lemon.exe](lemon.exe) вм просто его запускаете и у вас все работает,но есть единственное предупреждение:
у вас может орать антивирусник,что это какой то вирус,это нормально для файлов без какой либо репутации приятного пользования...(наверное)
